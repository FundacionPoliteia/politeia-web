Brand kit digital

Sistema base para remodelar productos, paneles y sitios que conviven bajo Politeia. Define colores, tipografia, componentes y reglas de uso para mantener una identidad editorial, clara y consistente.

Fuente de verdad

Los tokens viven en archivos separados para poder importarlos desde otras apps sin copiar estilos a mano.

CSS

Importar `brand/politeia-tokens.css` antes de los estilos de cada app. Es el camino directo para web-apps.

JSON

Usar `brand/politeia-tokens.json` para Figma, Style Dictionary o cualquier pipeline de design tokens.

Colores

La paleta se apoya en tinta, hueso y blanco. El azul senala accion. El rosa acompana: debe aparecer poco para conservar valor.

#1a1a37
Tinta--tinta

Texto principal, fondos oscuros y acciones de maxima jerarquia.

#137a9f
Azul--azul

Links, botones, iconos, filtros activos y foco de interaccion.

#fce3da
Rosa--rosa

Badges, detalles y acentos blandos. Nunca como fondo dominante.

#f7f5f2
Hueso--hueso

Fondo general. Calido, editorial y menos duro que el blanco puro.

#ffffff
Blanco--blanco

Tarjetas, formularios, modales y superficies elevadas.

#5b5b6b
Gris--gris

Texto secundario, metadatos, ayudas y descripciones.

Proporcion recomendada

Hueso 55%

Tipografia

Fraunces da tono editorial y presencia. Archivo sostiene lectura, paneles, tablas y controles. JetBrains Mono queda reservado para codigo y muestras tecnicas.

Componentes base

Patrones minimos para que las apps convivan sin perder identidad.

Botones

Accesibilidad

Combinaciones seguras para texto. Para parrafos largos, usar tinta o gris; reservar azul para accion o enfasis.

CombinacionUsoEstado
Tinta sobre hueso/blancoTexto principalSeguro
Gris sobre hueso/blancoTexto secundarioSeguro
Blanco sobre azulBotonesSeguro
Azul sobre huesoEtiquetas, links, iconosUsar con peso
Rosa como fondo dominanteSecciones completasEvitar

Codigo

Import recomendado para nuevas apps.

@import "./brand/politeia-tokens.css";

.app-shell {
  background: var(--color-page);
  color: var(--color-text);
  font-family: var(--texto);
}

.primary-action {
  background: var(--color-action);
  border-radius: var(--radius-pill);
  color: var(--blanco);
}
